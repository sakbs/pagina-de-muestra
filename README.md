# Página de Muestra

Este proyecto es una página web creada como ejemplo para mostrar a nuestros clientes, 
estilos, despliegue y control de versiones con Git y GitHub.
El objetivo es mostrar proyectos web y demostrar conocimientos en HTML, CSS, JavaScript y buenas prácticas de desarrollo.

---

## Tecnologías utilizadas

- **HTML5** — estructura del sitio  
- **CSS3** — estilos, diseño responsivo  
- **JavaScript** — interacción básica (si aplica)  
- **Bootstrap / Font Awesome** — componentes y estilos adicionales  
- **Git & GitHub** — control de versiones y despliegue
